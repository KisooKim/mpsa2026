# MPSA 2026 Program Viewer Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a static HTML+CSS+JavaScript viewer for the MPSA 2026 conference program that supports filtering by date, author, division, topic keyword, and session type, with inline session details, persistent filters, and named saved filter presets ("Saved Views").

**Architecture:** Single-page vanilla JavaScript app that loads a pre-parsed `data/program.json` (generated once by a Python script from manually-dumped HTML). Frontend modules are plain `<script>` files attaching to a global `MPSA` namespace. All state (filters, presets) persists in `localStorage`. No framework, no bundler, no runtime dependencies.

**Tech Stack:**
- Python 3 + `beautifulsoup4` + `lxml` for the one-time parser
- Vanilla HTML/CSS/JavaScript for the frontend (no bundler)
- `node:test` (Node ≥ 18 built-in) + `node:vm` for pure-JS module tests
- Python `unittest` for parser tests
- `python -m http.server` for local serving

## Prerequisites (completed before Task 2)

Data collection had to be split into two steps because the allacademic "Browse by Day" listing pages only contain shallow per-session metadata (title/time/division/type) — the chair, discussant, papers, and author data live on per-session detail pages. Fortunately the detail pages are publicly reachable via the `online_program_direct_link` endpoint (no session cookie needed).

**Both steps are already complete before Task 2 begins:**

1. **Day listing dumps (manual, 4 files)** — user dumped each day's HTML via `copy(document.documentElement.outerHTML)` in DevTools. Files: `raw_html/day1-2026-04-23.html` through `raw_html/day4-2026-04-26.html`. Session counts: 298, 363, 351, 87 (total 1,099).

2. **Bulk detail fetch (automated)** — `scripts/fetch_details.py` reads all 4 day listings, extracts the 1,099 numeric session IDs (filtering out the `session_result_N` duplicates), then fetches each session detail HTML via the `online_program_direct_link` URL with 10 parallel workers. Output: `raw_html/details/session_{id}.html` (~26 MB total, ~23 KB per session).

The parser from Task 3 onward operates **only on `raw_html/details/session_*.html`**. The listing files are not used by the parser — they served their purpose during ID extraction, and the detail pages' own headers contain date, time, room, session type, and division.

## Module Contracts (JS frontend)

Each module attaches one leaf to `window.MPSA`. Loaded in this order in `index.html`:
1. `storage.js` → `MPSA.storage`
2. `filters.js` → `MPSA.filters`
3. `search.js` → `MPSA.search`
4. `render.js` → `MPSA.render`
5. `app.js` (bootstrap; no namespace leaf)

**`MPSA.storage`** — pure `localStorage` wrapper, no DOM access.
- `loadFilters()` → `FilterState` (or default empty)
- `saveFilters(state)` → void
- `listPresets()` → `Preset[]`
- `savePreset(name, filters)` → `Preset` (new preset with generated id)
- `updatePreset(id, filters)` → `Preset`
- `renamePreset(id, newName)` → `Preset`
- `deletePreset(id)` → void
- `loadActivePresetId()` → `string | null`
- `saveActivePresetId(id | null)` → void

**`MPSA.filters`** — pure functions for filter state shape + matching.
- `emptyState()` → `FilterState` with all empty arrays and `keyword: ""`
- `isEmpty(state)` → boolean (no filter active)
- `matches(session, state)` → boolean (does this session pass all filters?)
- `equals(a, b)` → boolean (deep compare two FilterStates)

**`MPSA.search`** — pure functions for indexing + autocomplete.
- `buildPeopleIndex(sessions)` → `Array<{name, affiliation, normalized}>`
- `matchPeople(index, query, limit=8)` → `Array<{name, affiliation, match_ranges}>`

**`MPSA.render`** — DOM rendering; takes program.json + FilterState and mutates specific DOM containers.
- `renderMain(sessions, state)` → void (renders main content area: empty state OR day sections)
- `renderFilterSummary(state, totalCount, activePresetName)` → void
- `renderSidebar(program, state, presets, activePresetId)` → void

**`FilterState`** type (pseudo-schema):
```
{
  dates: string[],         // ISO dates like "2026-04-23"
  authors: string[],       // canonical names from program.json
  divisions: string[],     // division names
  sessionTypes: string[],  // session type strings
  keyword: string          // free text, lowercase normalization happens at match time
}
```

**`Preset`** type:
```
{
  id: string,           // e.g. "p_1712345678"
  name: string,
  filters: FilterState,
  createdAt: number,    // Date.now()
  updatedAt: number
}
```

```python
"""MPSA 2026 program parser — converts a day HTML dump into structured session dicts."""
from __future__ import annotations

import hashlib
import re
from typing import Optional

from bs4 import BeautifulSoup, Tag


# ---- selectors (adjust to match docs/superpowers/notes/parser-structure.md) ----
SESSION_SELECTOR = "div.session"  # <-- REPLACE with the real selector from Task 2 notes
TITLE_SELECTOR = "h3, .session-title"
TIME_SELECTOR = ".session-time, .time"
ROOM_SELECTOR = ".session-room, .room"
TYPE_SELECTOR = ".session-type, .type"
DIVISION_SELECTOR = ".session-division, .division"
CHAIR_SELECTOR = ".chair, .session-chair"
DISCUSSANT_SELECTOR = ".discussant, .session-discussant"
PAPER_SELECTOR = ".paper, .session-paper"
PAPER_TITLE_SELECTOR = ".paper-title, h4"
AUTHOR_SELECTOR = ".author, .paper-author"

TIME_RE = re.compile(
    r"(\d{1,2}):(\d{2})\s*(AM|PM)\s*[-–to]+\s*(\d{1,2}):(\d{2})\s*(AM|PM)",
    re.I,
)


def _text(el: Optional[Tag]) -> str:
    return el.get_text(" ", strip=True) if el else ""


def _to_24h(h: int, m: int, period: str) -> str:
    period = period.upper()
    if period == "PM" and h != 12:
        h += 12
    if period == "AM" and h == 12:
        h = 0
    return f"{h:02d}:{m:02d}"


def _parse_time_range(text: str) -> tuple[str, str, str]:
    """Return (start_24h, end_24h, human_slot). Empty strings if no match."""
    m = TIME_RE.search(text)
    if not m:
        return ("", "", "")
    sh, sm, sp, eh, em, ep = m.groups()
    start = _to_24h(int(sh), int(sm), sp)
    end = _to_24h(int(eh), int(em), ep)
    human = f"{int(sh)}:{sm} {sp.upper()}"
    return (start, end, human)


def _parse_person(node: Tag) -> dict:
    """Extract a name/affiliation pair from a participant element.

    Common formats:
      "Jane Smith (University of Chicago)"
      <span class="name">Jane Smith</span> <span class="affil">University of Chicago</span>
    """
    name_el = node.select_one(".name")
    affil_el = node.select_one(".affiliation, .affil, .institution")
    if name_el or affil_el:
        return {"name": _text(name_el), "affiliation": _text(affil_el)}
    raw = _text(node)
    m = re.match(r"^(.*?)\s*\(([^)]+)\)\s*$", raw)
    if m:
        return {"name": m.group(1).strip(), "affiliation": m.group(2).strip()}
    return {"name": raw, "affiliation": ""}


def _session_id(date_iso: str, start_time: str, room: str, title: str) -> str:
    payload = f"{date_iso}|{start_time}|{room}|{title}".encode("utf-8")
    return hashlib.sha1(payload).hexdigest()[:12]


def parse_sessions(html: str, date_iso: str) -> list[dict]:
    soup = BeautifulSoup(html, "lxml")
    sessions: list[dict] = []
    for node in soup.select(SESSION_SELECTOR):
        try:
            title = _text(node.select_one(TITLE_SELECTOR))
            if not title:
                continue
            time_text = _text(node.select_one(TIME_SELECTOR))
            start, end, human = _parse_time_range(time_text)
            room = _text(node.select_one(ROOM_SELECTOR))
            session_type = _text(node.select_one(TYPE_SELECTOR))
            division = _text(node.select_one(DIVISION_SELECTOR))

            chairs = [_parse_person(n) for n in node.select(CHAIR_SELECTOR)]
            discussants = [_parse_person(n) for n in node.select(DISCUSSANT_SELECTOR)]

            papers = []
            for p in node.select(PAPER_SELECTOR):
                papers.append({
                    "title": _text(p.select_one(PAPER_TITLE_SELECTOR)),
                    "authors": [_parse_person(a) for a in p.select(AUTHOR_SELECTOR)],
                })

            sessions.append({
                "id": _session_id(date_iso, start, room, title),
                "date": date_iso,
                "start_time": start,
                "end_time": end,
                "time_slot": human,
                "room": room,
                "title": title,
                "session_type": session_type,
                "division": division,
                "chair": chairs,
                "co_chair": [],  # populated in Task 4 if distinguishable
                "discussant": discussants,
                "papers": papers,
            })
        except Exception as e:
            print(f"[parse_sessions] skip session due to error: {e}", file=__import__("sys").stderr)
            continue
    return sessions
```

```python
from datetime import datetime, timezone


def build_program(sessions: list[dict]) -> dict:
    days = sorted({s["date"] for s in sessions if s.get("date")})
    divisions = sorted({s["division"] for s in sessions if s.get("division")})
    session_types = sorted({s["session_type"] for s in sessions if s.get("session_type")})
    return {
        "meta": {
            "conference": "MPSA 2026",
            "generated_at": datetime.now(timezone.utc).isoformat(),
            "days": days,
        },
        "divisions": divisions,
        "session_types": session_types,
        "sessions": sessions,
    }
```

```javascript
import { readFileSync } from "node:fs";
import { fileURLToPath } from "node:url";
import { dirname, resolve } from "node:path";
import vm from "node:vm";

const __dirname = dirname(fileURLToPath(import.meta.url));
const ROOT = resolve(__dirname, "..", "..");

export function loadMpsaModules(relativePaths, extraGlobals = {}) {
  const sandbox = {
    console,
    setTimeout,
    clearTimeout,
    Date,
    JSON,
    Math,
    Object,
    Array,
    String,
    Number,
    Boolean,
    RegExp,
    Error,
    Map,
    Set,
    ...extraGlobals,
  };
  sandbox.window = sandbox;
  sandbox.globalThis = sandbox;
  vm.createContext(sandbox);
  for (const rel of relativePaths) {
    const src = readFileSync(resolve(ROOT, rel), "utf8");
    vm.runInContext(src, sandbox, { filename: rel });
  }
  return sandbox;
}

export function mockLocalStorage() {
  const store = {};
  return {
    store,
    api: {
      getItem: (k) => (k in store ? store[k] : null),
      setItem: (k, v) => { store[k] = String(v); },
      removeItem: (k) => { delete store[k]; },
      clear: () => { Object.keys(store).forEach((k) => delete store[k]); },
      get length() { return Object.keys(store).length; },
      key: (i) => Object.keys(store)[i] ?? null,
    },
  };
}
```

```javascript
(function (root) {
  const NS = root.MPSA = root.MPSA || {};

  const KEY_FILTERS = "mpsa2026-filters";
  const KEY_PRESETS = "mpsa2026-presets";
  const KEY_ACTIVE  = "mpsa2026-active-preset";

  function emptyFilters() {
    return { dates: [], authors: [], divisions: [], sessionTypes: [], keyword: "" };
  }

  function safeParse(raw, fallback) {
    if (raw == null) return fallback;
    try { return JSON.parse(raw); } catch (_) { return fallback; }
  }

  function loadFilters() {
    const state = safeParse(root.localStorage.getItem(KEY_FILTERS), null);
    if (!state || typeof state !== "object") return emptyFilters();
    return {
      dates: Array.isArray(state.dates) ? state.dates : [],
      authors: Array.isArray(state.authors) ? state.authors : [],
      divisions: Array.isArray(state.divisions) ? state.divisions : [],
      sessionTypes: Array.isArray(state.sessionTypes) ? state.sessionTypes : [],
      keyword: typeof state.keyword === "string" ? state.keyword : "",
    };
  }

  function saveFilters(state) {
    root.localStorage.setItem(KEY_FILTERS, JSON.stringify(state));
  }

  function listPresets() {
    const arr = safeParse(root.localStorage.getItem(KEY_PRESETS), []);
    return Array.isArray(arr) ? arr : [];
  }

  function savePresetsArray(arr) {
    root.localStorage.setItem(KEY_PRESETS, JSON.stringify(arr));
  }

  function newId() {
    return "p_" + Date.now().toString(36) + "_" + Math.floor(Math.random() * 1e6).toString(36);
  }

  function savePreset(name, filters) {
    const now = Date.now();
    const preset = { id: newId(), name: String(name), filters, createdAt: now, updatedAt: now };
    const arr = listPresets();
    arr.push(preset);
    savePresetsArray(arr);
    return preset;
  }

  function updatePreset(id, filters) {
    const arr = listPresets();
    const idx = arr.findIndex((p) => p.id === id);
    if (idx < 0) throw new Error("preset not found: " + id);
    arr[idx] = { ...arr[idx], filters, updatedAt: Date.now() };
    savePresetsArray(arr);
    return arr[idx];
  }

  function renamePreset(id, newName) {
    const arr = listPresets();
    const idx = arr.findIndex((p) => p.id === id);
    if (idx < 0) throw new Error("preset not found: " + id);
    arr[idx] = { ...arr[idx], name: String(newName), updatedAt: Date.now() };
    savePresetsArray(arr);
    return arr[idx];
  }

  function deletePreset(id) {
    const arr = listPresets().filter((p) => p.id !== id);
    savePresetsArray(arr);
  }

  function loadActivePresetId() {
    const raw = root.localStorage.getItem(KEY_ACTIVE);
    return raw || null;
  }

  function saveActivePresetId(id) {
    if (id == null) root.localStorage.removeItem(KEY_ACTIVE);
    else root.localStorage.setItem(KEY_ACTIVE, String(id));
  }

  NS.storage = {
    emptyFilters, loadFilters, saveFilters,
    listPresets, savePreset, updatePreset, renamePreset, deletePreset,
    loadActivePresetId, saveActivePresetId,
  };
})(typeof window !== "undefined" ? window : globalThis);
```

```javascript
(function (root) {
  const NS = root.MPSA = root.MPSA || {};

  function emptyState() {
    return { dates: [], authors: [], divisions: [], sessionTypes: [], keyword: "" };
  }

  function isEmpty(state) {
    if (!state) return true;
    return (
      (state.dates || []).length === 0 &&
      (state.authors || []).length === 0 &&
      (state.divisions || []).length === 0 &&
      (state.sessionTypes || []).length === 0 &&
      !(state.keyword && state.keyword.trim())
    );
  }

  function eqArr(a, b) {
    if (a.length !== b.length) return false;
    for (let i = 0; i < a.length; i++) if (a[i] !== b[i]) return false;
    return true;
  }

  function equals(a, b) {
    return (
      eqArr(a.dates || [], b.dates || []) &&
      eqArr(a.authors || [], b.authors || []) &&
      eqArr(a.divisions || [], b.divisions || []) &&
      eqArr(a.sessionTypes || [], b.sessionTypes || []) &&
      (a.keyword || "") === (b.keyword || "")
    );
  }

  function collectSessionText(session) {
    const parts = [session.title || ""];
    for (const p of session.papers || []) {
      parts.push(p.title || "");
      for (const a of p.authors || []) parts.push(a.name || "");
    }
    for (const c of session.chair || []) parts.push(c.name || "");
    for (const c of session.co_chair || []) parts.push(c.name || "");
    for (const d of session.discussant || []) parts.push(d.name || "");
    return parts.join(" ").toLowerCase();
  }

  function matchesDates(session, state) {
    if (!state.dates || state.dates.length === 0) return true;
    return state.dates.includes(session.date);
  }

  function matchesDivisions(session, state) {
    if (!state.divisions || state.divisions.length === 0) return true;
    return state.divisions.includes(session.division);
  }

  function matchesSessionTypes(session, state) {
    if (!state.sessionTypes || state.sessionTypes.length === 0) return true;
    return state.sessionTypes.includes(session.session_type);
  }

  function matchesAuthors(session, state) {
    if (!state.authors || state.authors.length === 0) return true;
    const bag = (session.all_people || []).join(" | ");
    for (const authorQuery of state.authors) {
      const needle = String(authorQuery).toLowerCase();
      if (bag.includes(needle)) return true;
    }
    return false;
  }

  function matchesKeyword(session, state) {
    const kw = (state.keyword || "").trim().toLowerCase();
    if (!kw) return true;
    const haystack = collectSessionText(session);
    const tokens = kw.split(/\s+/).filter(Boolean);
    for (const t of tokens) {
      if (!haystack.includes(t)) return false;
    }
    return true;
  }

  function matches(session, state) {
    return (
      matchesDates(session, state) &&
      matchesDivisions(session, state) &&
      matchesSessionTypes(session, state) &&
      matchesAuthors(session, state) &&
      matchesKeyword(session, state)
    );
  }

  NS.filters = { emptyState, isEmpty, equals, matches };
})(typeof window !== "undefined" ? window : globalThis);
```

```javascript
(function (root) {
  const NS = root.MPSA = root.MPSA || {};

  function collectPeopleFromSession(session, bag) {
    const push = (p) => {
      if (!p || !p.name) return;
      const key = (p.name + "|" + (p.affiliation || "")).toLowerCase();
      if (!bag.has(key)) {
        bag.set(key, { name: p.name, affiliation: p.affiliation || "", normalized: p.name.toLowerCase() });
      }
    };
    for (const c of session.chair || []) push(c);
    for (const c of session.co_chair || []) push(c);
    for (const d of session.discussant || []) push(d);
    for (const paper of session.papers || []) {
      for (const a of paper.authors || []) push(a);
    }
  }

  function buildPeopleIndex(sessions) {
    const bag = new Map();
    for (const s of sessions) collectPeopleFromSession(s, bag);
    return Array.from(bag.values()).sort((a, b) => a.name.localeCompare(b.name));
  }

  function matchPeople(index, rawQuery, limit = 8) {
    const q = (rawQuery || "").trim().toLowerCase();
    if (!q) return [];
    const results = [];
    for (const p of index) {
      const i = p.normalized.indexOf(q);
      if (i >= 0) {
        results.push({ name: p.name, affiliation: p.affiliation, matchStart: i, matchEnd: i + q.length });
        if (results.length >= limit) break;
      }
    }
    return results;
  }

  NS.search = { buildPeopleIndex, matchPeople };
})(typeof window !== "undefined" ? window : globalThis);
```

# MPSA 2026 Program Viewer

A local HTML/CSS/JavaScript viewer for the 83rd Annual Midwest Political Science Association Conference (April 23–26, 2026). Filter sessions by date, author, division, topic keyword, or session type, see parallel sessions side-by-side in a time-slot timeline, and save filter combinations as "Saved Views" for later.

## What it does

- **Filter** MPSA sessions by date, author, division, topic keyword, session type
- **Timeline view** — see same-time parallel sessions side-by-side, grouped by day
- **Inline details** — click a session to see chair, discussant, papers, and authors without losing context
- **Saved Views** — name and switch between filter presets (e.g. "American Politics Track", "My friends' panels")
- **Persistent** — filters and presets persist in localStorage across reloads
